This document assumes that the reader is capable of understanding English language text and has basic computer literacy.

📥 Adding Resources

In order to use the Portal app, you will need content to view within it. Inside the app, content is referred to as resources.

The app can display two different types of resources: 📚 Courses and 🗺️ Course Maps.

📚 Courses are a collection of textbooks. Textbooks can be 📕 completable, where you can track your progress, or they can be 📒 non-completable. Non-completable textbooks are typically included as reference materials or supplemental reading.

🗺️ Course Maps are an interactive tree of 📚 Courses. These help you plan your learning journey by providing an intended order to complete courses in.

🎨 Creating Resources

⚠ Unlke importing existing resources, creating resources for Portal is a more involved progress. A degree of specialized technical knowledge is required.

You will need to understand the following concepts:

  • TOML configuration format
  • Universally Unique Identifier (UUID)
  • JSON schema
  • EPUB e-Book format
    • Extensible Markup Language (XML)
    • Hypertext Markup Language (HTML)
    • OCF abstract container (EPUB-specific file hierarchy)
    • EPUB navigation document (EPUB v3)
    • Navigation Center eXtended (EPUB v2)
  • ZIP archive format

A code editor (such as Visual Studio Code) is strongly recommended, along with a tool or plugin to validate TOML files against a JSON schema.

A specialized ePub editor (such as Sigil) is recommended. However, in a pinch, an ordinary code editor can be used instead.
