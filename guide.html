âš  Unlke importing existing resources, creating resources for Portal is a more involved progress. A degree of specialized technical knowledge is required.

You will need to understand the following concepts:

  • TOML configuration format
  • Universally Unique Identifier (UUID)
  • JSON schema
  • EPUB e-Book format
    • Extensible Markup Language (XML)
    • Hypertext Markup Language (HTML)
    • OCF abstract container (EPUB-specific file hierarchy)
    • EPUB navigation document (EPUB v3)
    • Navigation Center eXtended (EPUB v2)
  • ZIP archive format

A code editor (such as Visual Studio Code) is strongly recommended, along with a tool or plugin to validate TOML files against a JSON schema.

A specialized ePub editor (such as Sigil) is recommended. However, in a pinch, an ordinary code editor can be used instead.
