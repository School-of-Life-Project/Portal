🏫 Portal

Portal is an app built for independent learners.

Unlike most eLearning applications, Portal is focused on making use of existing learning materials. Any textbook or set of textbooks can be converted into a Portal course, as long as it is available in the EPUB format.

Portal is designed to perform well on low-end hardware and does not require an internet connection to use.

📥 Installing

🚧 Portal is a work in progress. Please report any issues you find.

 Windows

⚠ This installer may require an internet connection.

 Download Portal v0.3.9 Installer (64-bit x86)

Follow the installer's prompts to install the application.

Offline Installation

If the device you are installing the application on does not have an internet connection, you will need to install the WebView2 Runtime before running the installer.

Windows 10 (version 1803 and later) and 11 come with WebView2 preinstalled.


 macOS

 Download Portal v0.3.9 Installer (Intel + Apple Silicon)

Use the disk image to install the application to the /Applications folder.

Then, open a Terminal window and run the following command:

xattr -d com.apple.quarantine /Applications/Portal.app

🐧 Linux

Installation instructions are distribution specific.

If your distribution is not yet listed, you will need to build the application from source code.

DPKG-based distros (Debian, Ubuntu, etc...)

RPM-based distros (Red Hat, Fedora, AlmaLinux, etc...)

🧑‍🎓 Using Portal

Portal has a built-in user guide, which guides the user through post-installation setup and usage.

This guide can be accessed using the 📜 Guide button on the application's home screen.
